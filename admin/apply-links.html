How to Update Links

Individual Updates:

  1. Click the edit button next to any card
  2. Enter the new official bank application URL
  3. Click save to update the link
  4. Changes are reflected immediately across the website

Bulk Updates:

  1. Use the bank filter to select a specific bank
  2. Update the pattern in data/apply-links.ts
  3. All cards from that bank will be updated
  4. Export CSV to backup current links
